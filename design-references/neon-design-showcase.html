5D Method

五维升维法

🎭
主题感官化
将抽象主题转化为可感知的视觉符号，用色彩、形状、动效传递情绪，让观众"感觉"到内容而非仅仅"阅读"内容。
🎬
剧场式切分
每页PPT都是一个"场景"，有开场、高潮、收尾。用视觉节奏控制观众注意力，像电影一样讲述故事。
📱
发布会级排版
借鉴苹果发布会的极简美学：大字、大图、大量留白。每页只讲一件事，但讲得震撼。
🧩
语义组件降维
将复杂信息拆解为语义组件：card-rise（上升）、card-fall（坠落）、warning-box（警示）、ok-box（成功），让信息自带情绪。
🎯
焦点入场导演
控制每个元素的入场时机和方式：淡入、滑入、缩放、闪烁。让观众的视线跟随你的节奏移动。
✨
霓虹光效系统
建立可复用的光效库：文字发光、边框发光、背景光晕、粒子动画。一键应用，风格统一。

CSS Code

可复用代码

/* 霓虹赛博朋克配色系统 */
:root {
  --bg: #050508;
  --neon-cyan: #00ffff;
  --neon-pink: #ff00ff;
  --neon-purple: #8b5cf6;
  --glow-cyan: rgba(0, 255, 255, 0.7);
}

/* 霓虹文字发光效果 */
.neon-text {
  color: var(--neon-cyan);
  text-shadow: 
    0 0 5px var(--glow-cyan),
    0 0 10px var(--glow-cyan),
    0 0 20px var(--glow-cyan),
    0 0 40px var(--glow-cyan);
}

/* 霓虹边框发光效果 */
.neon-border {
  border: 1px solid var(--neon-cyan);
  box-shadow: 
    0 0 5px var(--glow-cyan),
    0 0 10px var(--glow-cyan),
    inset 0 0 5px var(--glow-cyan);
}

/* 渐变文字动画 */
.gradient-text {
  background: linear-gradient(90deg, #00ffff, #ff00ff);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  animation: gradientShift 3s ease infinite;
}
