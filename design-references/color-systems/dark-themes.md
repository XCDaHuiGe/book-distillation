# 深色主题配色系统

> 适合发布会、沉浸式阅读、高端展示场景

---

## 配色原则

1. **背景不能纯黑**：必须带微妙色相（暖黑/冷黑）
2. **主强调色唯一**：一套配色只用一个主强调色
3. **语义色固定**：红=负向/警告，绿=正向/成功

---

## 配色方案库

### 方案1：天道金（适合哲学/文学）

```css
:root {
  --bg: #0a0a0b;        /* 暖黑 */
  --bg-elevated: #141416;
  --fg: #f5f5f7;
  --fg-muted: #8a8a8e;
  --accent: #c9a227;    /* 金色 */
  --accent-glow: rgba(201, 162, 39, 0.3);
  --rise: #10b981;      /* 上升/正向 */
  --fall: #ef4444;      /* 坠落/负向 */
}
```

**适合场景**：遥远的救世主、天道、哲学类内容
