## 作品5：2025-2026 作品集

**作者**：跳跃的小鱼  
**类型**：UI/交互/动效设计  
**链接**：https://www.zcool.com.cn/work/ZNzM1OTAzNTI=.html

### 设计要点

| 维度 | 分析 |
|:---|:---|
| **配色** | **暗黑色系 + 霓虹强调色**，赛博朋克风格 |
| **排版** | 大图展示，视觉冲击力强 |
| **组件** | 霓虹边框、发光效果、渐变背景 |
| **风格** | **霓虹科技、暗黑色系、赛博朋克** |

### 可复用设计元素

```css
/* 赛博朋克配色 */
:root {
  --bg: #0a0a0f;
  --fg: #e5e7eb;
  --neon-pink: #ff00ff;
  --neon-blue: #00ffff;
  --neon-purple: #8b5cf6;
  --glow-pink: rgba(255, 0, 255, 0.5);
  --glow-blue: rgba(0, 255, 255, 0.5);
}

/* 霓虹边框效果 */
.neon-border {
  border: 1px solid var(--neon-blue);
  box-shadow: 0 0 10px var(--glow-blue),
              0 0 20px var(--glow-blue),
              0 0 40px var(--glow-blue);
}
```

**PPT 应用**：适合科技、游戏、未来主题内容

## 学习总结

### 提取的设计风格

| 风格 | 配色特点 | 适用场景 |
|:---|:---|:---|
| **霓虹科技风** | 暗黑 + 霓虹粉/蓝/紫 | 科技、游戏、未来主题 |
| **数据可视化** | 深蓝 + 多色数据点 | 数据报告、分析类 |
| **HMI科技风** | 深黑 + 青色强调 | 汽车、工业、技术 |
| **赛博朋克** | 暗黑 + 霓虹边框发光 | 创意、艺术、游戏 |
| **专业商务风** | 浅色/深蓝 + 蓝色强调 | 商业、工具、企业 |

### PPT 设计可复用元素

1. **霓虹边框效果**：用于强调卡片/组件
2. **发光效果**：用于金句/重点文字
3. **数据配色系统**：用于图表/数据展示
4. **暗黑科技风**：用于科技/未来主题内容
