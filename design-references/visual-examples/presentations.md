# 发布会/演讲 PPT 案例学习

> 从 Apple、Huawei、Google 等发布会中学习顶级 PPT 设计

## 学习要点提取模板

每个案例按以下维度分析：

| 维度 | 分析内容 |
|:---|:---|
| **配色** | 主色、辅色、背景色、语义色 |
| **排版** | 字号层级、字重对比、留白比例 |
| **组件** | 卡片、数据展示、时间线样式 |
| **动画** | 入场顺序、延迟时间、切换效果 |
| **节奏** | 信息密度、页面数量、呼吸感 |

## 案例库

### 案例1：Apple Keynote

**来源**：Apple 产品发布会

**配色方案**：
```css
--bg: #000000;
--fg: #ffffff;
--accent: #f5f5f7;
--highlight: #0071e3;
```

**排版特点**：
- 标题：96px / 900weight / 全大写
- 正文：24px / 400weight
- 对比度：1:4 以上

**动画特点**：
- 标题先出，正文后出
- 延迟：0.2s 递进
- 切换：淡入淡出

**可复用代码**：
```html
<!-- Apple 风格标题 -->
<h1 class="apple-title">THINK DIFFERENT</h1>

<style>
.apple-title {
  font-size: clamp(64px, 8vw, 120px);
  font-weight: 900;
  letter-spacing: -0.02em;
  color: #f5f5f7;
}
</style>
```

**来源**：Huawei 产品发布会

```css
--bg: #0a0a0a;

```

**排版特点**：
- 中国风衬线标题
- 数据展示用大数字 + 小标签
- 渐变背景增加层次
