霓虹赛博朋克配色

:root { --bg: #0a0a0f; --neon-blue: #00ffff; --neon-pink: #ff00ff; --neon-purple: #8b5cf6; --glow-cyan: rgba(0, 255, 255, 0.5); --glow-pink: rgba(255, 0, 255, 0.5); }

霓虹文字效果

.neon-text { color: var(--neon-blue); text-shadow: 0 0 5px var(--glow-cyan), 0 0 10px var(--glow-cyan), 0 0 20px var(--glow-cyan); }

霓虹边框效果

.neon-border { border: 1px solid var(--neon-blue); box-shadow: 0 0 5px var(--glow-cyan), 0 0 10px var(--glow-cyan), 0 0 20px var(--glow-cyan); }

PPT 应用建议

科技/AI内容
使用霓虹赛博朋克配色，视觉冲击力强，符合科技感
汽车/硬件
使用HMI科技风，专业感强，青色强调醒目
数据报告
使用数据可视化配色，多色区分不同数据类型
文学/哲学
使用天道金配色，暖黑背景 + 金色强调，庄重典雅
