# 主题色预设（Themes）

## 氛围系统（V7.0 核心概念）

**不要只选颜色，要选"氛围系统"。** 提取内容的精神内核，映射为色温、光影和材质。

### 氛围系统三要素

1. **深色背景**（非纯黑，带微妙色相）— 营造空间感
2. **1个主强调色**（如金色/琥珀色）— 金句、重点、标题
3. **2个语义色**（如红=坠落/绿=上升）— 逻辑关系可视化

### 禁止事项

- ❌ 禁止使用任何默认蓝/紫色系
- ❌ 禁止纯黑背景（`#000000`），必须带微妙色相
- ❌ 禁止用户自定义颜色——色彩搭配错了画面瞬间变丑

### 示例：《遥远的救世主》氛围系统

| 元素 | 颜色 | 语义 |
|:---|:---|:---|
| 深色背景 | `#0a0a0b`（带微暖的黑） | 未知、深邃 |
| 主强调色 | `#c9a227`（金色） | 天道、规律 |
| 语义色1 | `#ef4444`（暗红） | 人性、坠落 |
| 语义色2 | `#10b981`（翠绿） | 救赎、上升 |

## 使用方法

1. 问用户选哪套（或基于内容推荐一套）
2. 打开 `assets/template.html` 的 `<style>` 块
3. 找到开头的 `:root{` 块
4. **整体替换**以下变量：
   - `--ink` / `--ink-rgb` — 墨色（深色）
   - `--paper` / `--paper-rgb` — 纸色（浅色）
   - `--paper-tint` — 纸色加深
   - `--ink-tint` — 墨色减淡
   - `--accent` / `--accent-rgb` — 强调色

## 🖋 墨水经典（Monocle 默认）

**适合**：通用分享、商业发布、科技产品、任何场景都安全的默认选择。

**调性**：纯墨黑 + 暖米白，杂志感最强，Monocle / Apricot / A Book Apart 风。

```css
--ink:#0a0a0b;
--ink-rgb:10,10,11;
--paper:#f1efea;
--paper-rgb:241,239,234;
--paper-tint:#e8e5de;
--ink-tint:#18181a;
--accent:#c9a227;
--accent-rgb:201,162,39;
```

## 🌊 靛蓝瓷（Indigo Porcelain）

**适合**：科技/研究/数据分享、工程师文化、深度内容、技术发布会。

**调性**：深靛蓝 + 瓷白，冷静、理性、有深度，像学术期刊或蓝印花瓷器。

```css

--paper:#f1f3f5;
--paper-rgb:241,243,245;
--paper-tint:#e4e8ec;

```

--paper-tint:#ece7da;

## 切换原则

- **一份 deck 只用一套主题**，不要中途换色
- 粒子背景颜色跟随 `--accent` 变化
- `currentColor` 驱动的 border / icon 会跟随 section 的 text color 自动适配
- 选定主题后，`<title>` 文字和 `chrome` 文案可以强化该主题的语义

---

## ❌ 不要做的事

- ❌ **不允许混搭**（例如 ink 取墨水经典的，paper 取沙丘的）——会彻底违和
- ❌ **不允许用户随便给一个 hex 值**——需委婉拒绝并展示 5 套预设让选
- ❌ **不要直接修改 template.html 其他地方的颜色**——所有散落 rgba 都走 var，改 :root 一处即可
