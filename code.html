RESTORATION

范式B：体验复原（保底式）

适用类型
叙事虚构型 / 诗歌散文型 / 叙事非虚构型
信息密度弥散，100%段落共同编织体验网，删一细节可能断一条伏笔线
种子结构
叙事种子（3000-5000 Token）— 比逻辑种子宽60%

Paradigm C+D08

RECONSTRUCTION + CONTEXT

范式C：路径重建 + 范式D：语境保留

范式C 路径重建（跟踪式）

适用：思想哲学型 / 学术体系型

  • 辩证种子（3000-5000 Token）：核心追问 + 辩证结构 + 概念创造清单 + 反驳预见 + 思维路径图
  • 种子关注"作者如何一步步逼近问题"，不是"回答了什么"
  • 推导链条不可跳步 — "因为A且B所以C"不能压缩为"C"
  • 螺旋结构禁止压平为线性 — 必须保留每轮辩证的新维度
  • Agent分工：思维追踪者 + 概念考古 + 策展人

范式D 语境保留（标注式）

适用：对话访谈型

  • 声音种子（2000-3000 Token）：对话者档案 + 立场差异图 + 语境标记 + 即兴洞见索引
  • 每个洞见必须附带"在什么对话语境中产生的"
  • 即兴表达的语言风格本身是信息 — 不可改写
  • 必须保留立场差异 — 不同对话者对同一问题的不同态度
  • Agent分工：语境捕手 + 洞见提取者 + 策展人
范式C的核心：读者要能重走作者的思维路径，不是只看终点。范式D的核心：洞见不是孤立的，语境是它的一部分。

TYPE DETECTION LOGIC

类型判定决策树

Step 0 全局扫描后追加类型判定：

对话占比 > 40%? → 是 → 对话访谈型 → 范式D

描写句占比 > 50% 且有虚构标记? → 是 → 叙事虚构型 → 范式B

章节标题含"第X章"且无明确论点? → 是 → 叙事非虚构型 → B+C混合

同一概念在不同章节反复深化? → 是 → 思想哲学型 → 范式C

概念层级清晰且有推导链条? → 是 → 学术体系型 → 范式C

大量类比且有认知升级路径? → 是 → 科普通识型 → A+C混合

语言本身即价值且改写即损坏? → 是 → 诗歌散文型 → 范式B(特殊)

条目独立且无递进关系? → 是 → 工具参考型 → 不蒸馏，改索引

有叙事壳包裹论点核? → 是 → 混合跨界型 → 先判壳核

默认 → 论说实用型 → 范式A

判定维度：文本结构特征（章节标题类型）/ 语言风格特征（论断/描写/对话占比）/ 信息密度分布 / 章节间关系（递进/推进/螺旋/独立）

Seed Architecture11

SEED STRUCTURE UPGRADE

四种种子结构

叙事种子（范式B）

  • 核心冲突轴 — 主冲突+次冲突+冲突根源
  • 人物关系速查 — 欲望/矛盾/弧线方向/转折章节
  • 情节线索网 — 主线+支线起承转合+交汇点
  • 关键意象表 — 首次出现/演变轨迹/指向主题
  • 叙事节奏图 — 张力分布+高潮低谷章节位置

辩证种子（范式C）

  • 核心追问 — 在追查什么（不是回答了什么）
  • 辩证结构 — 每轮正题/反题/合题/新维度
  • 概念创造清单 — 为何发明/替代了什么/边界
  • 反驳预见 — 预见的反对意见+如何回应
  • 思维路径图 — 追问起点→转向→突破→立场

体系种子（范式C）

  • 知识树结构 — 根→分支→叶，完整层级
  • 前置知识图 — 读懂第N章需先懂什么
  • 推导链条清单 — 起点→中间步骤→终点
  • 概念边界表 — 定义/什么不是它/前提条件
  • 例证索引 — 概念/原书例证/例证类型

声音种子（范式D）

  • 对话者档案 — 表达习惯/思维偏好/价值立场
  • 立场差异图 — 同一问题的不同态度
  • 语境标记 — 触发洞见的对话场景
  • 即兴洞见索引 — 洞见/产生语境/人物
  • 声音档案 — 每人的表达习惯和思维偏好

Paradigm B Agents13

FICTION AGENT TRIO

范式B：小说专属 3-Agent

Alpha → 叙事分析师

M1 叙事内核 — 驱动全书的叙事动力（替代一句话逻辑）

M2 人物谱系与弧线 — 每个核心角色的完整弧线+关系网+关键场景（替代核心知识地图）

M3 情节线索网 — 主线+支线起承转合+交汇点+伏笔回收（新增）

Beta → 场景考古学家

M4 关键场景复原 — 5-8个决定性场景的详细还原（替代落地行动指南）

每个场景卡片包含：位置/人物/叙事功能/原文对话节选/人物内心层/隐喻层/对全局的影响/伏笔追踪/删掉测试

M5 主题层叠 — 多重主题如何通过叙事交织呈现（替代颠覆性认知）

M6 意象与隐喻体系 — 反复出现的象征及其演变轨迹（新增）

Gamma → 策展人

M7 金句与遗珠 — 金句附场景上下文+被主线遮蔽的支线光芒（合并原M5+M6）

M8 叙事边界 — 这部小说的"不可翻译之处"，什么丢失了故事就塌了（替代边界约束）

核心转变

旧：3-Agent按功能拆解（结构/行动/策展）
新：3-Agent按叙事层次拆解（动力链/场景层/边界层）

小说需要的是一个完整叙事被不同视角照亮，不是被不同功能拆解。
